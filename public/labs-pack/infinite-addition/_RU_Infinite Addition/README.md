# 🧮 Лабораторная работа: Сложение трех чисел в столбик

Интерактивная веб-страница для обучения школьников сложению трех чисел в столбик с визуализацией переноса разрядов.

## 🎯 Цель проекта

Помочь ученикам понять и освоить:
- Сложение трех чисел в столбик
- Механизм переноса разрядов при сложении
- Пошаговый алгоритм выполнения сложения
- Визуальное представление процесса вычислений

## ✨ Основные возможности

### 📊 Регулировка сложности
- Выбор количества разрядов (2-5 разрядов)
- Автоматическая генерация примеров
- Постепенное увеличение сложности

### 👁️ Визуализация переноса
- Интерактивное отображение переносимых чисел
- Возможность показать/скрыть переносы
- Цветовое выделение переносимых разрядов

### ✅ Проверка и обучение
- Интерактивный ввод ответа по разрядам
- Мгновенная проверка правильности
- Подробное пошаговое объяснение решения
- Статистика успеваемости

### 📈 Статистика
- Количество решенных примеров
- Процент правильных ответов
- Отслеживание прогресса обучения

## 🚀 Как использовать

1. **Откройте файл `index.html` в веб-браузере**
2. **Выберите количество разрядов** в выпадающем списке
3. **Нажмите "Сгенерировать новый пример"** для создания задачи
4. **Решите пример**, вводя цифры в поля для ответа
5. **Нажмите "Проверить ответ"** для проверки
6. **Используйте "Показать решение"** для подробного объяснения
7. **Нажмите "👁️ Показать переносы"** для визуализации переносимых чисел
