# 🧮 Lab Work: Multiplication

Interactive web page for learning multiplication by elementary school students.

## 📋 Description

This project is a complete mathematics lab work that includes:

### 📚 Theory Section
- Explanation of multiplication concept
- Components of multiplication (multiplicand, multiplier, product)
- Properties of multiplication
- Multiplication table (part)

### 🎯 Practice Section
- **Exercise 1**: Simple multiplication with answer checking
- **Exercise 2**: Multiplication with visualization (dot grid)
- **Exercise 3**: Fill in the blanks in multiplication table
- Button to generate new exercises

### 📝 Test Section
- 10-question test with multiple choice answers
- Automatic result checking
- Knowledge assessment with percentage
- Option to retake the test

## 🚀 Running the Project

1. Download all project files
2. Open the `index.html` file in any modern browser
3. Start learning from the theory section
