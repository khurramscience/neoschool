# 🔢 Learning Number Place Values

Interactive web page for learning number place values, created for school students.

## 📚 Description

This project is an educational web page that helps school students understand how number place values work. The page includes:

- **Interactive number analysis** - enter any number and see its breakdown by place values
- **Practice exercises** - generate random numbers for training
- **Theoretical material** - explanation of place value concepts with examples
- **Modern design** - beautiful and user-friendly interface

## 🚀 How to Use

1. Open the `index.html` file in any modern browser
2. Enter a number in the input field and click "Analyze Number"
3. Study the result of the breakdown by place values
4. Try practice exercises by clicking "Generate Number"
5. Study the theoretical material at the bottom of the page

## ✨ Features

### 📊 Number Analysis
- Enter any positive number
- Get visual breakdown by place values
- See the value of each digit depending on its position

### 🎯 Practice Exercises
- Generation of random numbers for training
- Interactive input fields for each place value
- Instant answer checking
- Highlighting of correct and incorrect answers

### 📖 Theory
- Explanation of place value concepts
- List of basic place values
- Detailed example of number breakdown
