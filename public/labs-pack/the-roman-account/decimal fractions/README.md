# 🔢 Lab Work: Decimal Fractions

Interactive web page for learning decimal fractions by school students.

## 📚 Description

This lab work will help students understand the concept of decimal fractions through:
- **Theoretical material** with examples
- **Interactive visualization** of fractions
- **Practice exercises** for reinforcement
- **Educational game** for knowledge testing

## 🎯 Features

### 📖 Theory
- Explanation of decimal fraction concept
- Examples of converting ordinary fractions to decimals
- Visual representation of decimal number place values

### 🎨 Visualization
- Interactive 10x10 grid for visual representation of fractions
- Input any decimal fraction from 0 to 1
- Automatic display as ordinary fraction and percentages
- Color highlighting of filled cells

### ✏️ Practice
**Exercise 1: Fraction Conversion**
- Converting ordinary fractions to decimals
- Instant answer checking
- Detailed feedback

**Exercise 2: Fraction Comparison**
- Comparing two decimal fractions
- Choosing the correct comparison sign (<, =, >)
- Explanation of correct answer

### 🎮 Game
- Finding the correct decimal fraction among options
- Difficulty level system
- Score tracking
- Automatic level progression upon successful completion

## 🚀 Launch
