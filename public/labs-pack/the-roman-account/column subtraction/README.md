# 🧮 Laboratory Work: Column Subtraction

Interactive web laboratory work for schoolchildren to study column subtraction.

## 📋 Description

This laboratory work consists of three main blocks:

### 📚 Theory
- Detailed explanation of column subtraction principles
- Step-by-step rules for performing operations
- Visual example with detailed analysis
- Important points and common mistakes

### ✏️ Practice
- Generation of examples with adjustable difficulty
- Ability to adjust the number of digits (from 2 to 6)
- Instant answer checking
- Detailed step-by-step solution for errors
- Performance statistics

### 📝 Testing
- Assessment testing with configurable parameters
- From 5 to 20 questions
- Progress bar
- Detailed results analysis
- Grading system

## 🎯 Features

- **Safety**: All examples are generated so that the result is not negative
- **Whole numbers**: Work only with whole numbers (no decimal fractions)
- **Responsiveness**: Modern design adapted for mobile devices
- **Interactivity**: Instant feedback and detailed explanations

## 🚀 Launch

1. Download all files to one folder
2. Open the `index.html` file in any modern browser
3. Start learning from the "Theory" block
