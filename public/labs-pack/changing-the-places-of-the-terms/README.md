# 🔢 Laboratory Work: Commutative Property of Multiplication

Interactive web page for learning the commutative property of multiplication in the format of a laboratory work for school students.

## 📚 Description

This laboratory work helps students understand and reinforce the commutative property of multiplication, which states that **a × b = b × a**. The project consists of three main blocks:

### 1. 📖 Theory
- Explanation of the commutative property of multiplication
- Mathematical formula: a × b = b × a
- Visual examples with calculations
- Visual representation using dots

### 2. ✏️ Practice
- **Task 1**: Interactive calculator for checking the property
- **Task 2**: Creating your own examples with automatic verification

### 3. 🧪 Knowledge Check
- Test with 10 questions
- Automatic correctness checking
- Detailed results with explanations
- Possibility to retake the test

## 🚀 How to run

1. Download all project files
2. Open the `index.html` file in any modern browser
3. Start learning from the "Theory" section
