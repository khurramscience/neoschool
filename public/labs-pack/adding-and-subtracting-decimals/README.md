# 🔢 Лабораторная работа: Сложение и вычитание десятичных дробей

Интерактивная веб-страница для изучения сложения и вычитания десятичных дробей. Разработана для школьников с целью закрепления навыков работы с десятичными числами.

## ✨ Возможности

- **Два режима работы** - режим обучения и режим проверки знаний
- **Генерация случайных заданий** - автоматическое создание примеров с десятичными дробями
- **Интерактивная проверка** - мгновенная проверка ответов с подробными объяснениями
- **Пошаговое решение** - детальное объяснение процесса сложения и вычитания с визуализацией переноса разрядов
- **Режим тестирования** - 10 вопросов с оценкой и детальным анализом результатов
- **Статистика прогресса** - отслеживание правильных ответов и процента успешности
- **Экспорт результатов** - возможность скачать результаты тестирования
- **Адаптивный дизайн** - работает на компьютерах, планшетах и телефонах
- **Современный интерфейс** - красивый и понятный дизайн

## 🚀 Как использовать

### Режим обучения
1. Откройте файл `index.html` в любом современном браузере
2. Выберите "Режим обучения" (по умолчанию)
3. Введите ответ в поле ввода
4. Нажмите кнопку "Проверить" или клавишу Enter
5. Получите мгновенную обратную связь
6. При ошибке автоматически показывается пошаговое решение
7. Используйте кнопки "Предыдущий шаг" и "Следующий шаг" для изучения процесса
8. Нажмите "Показать решение" в любой момент для изучения алгоритма
9. Новое задание появится автоматически через 3 секунды

### Режим проверки знаний
1. Нажмите кнопку "Режим проверки"
2. Пройдите тест из 10 вопросов
3. Получите оценку и детальный анализ результатов
4. Скачайте результаты в формате JSON для сохранения
5. Используйте "Пройти тест заново" для повторного тестирования

## 🎯 Особенности

### Генерация заданий
- Случайные десятичные дроби от 0.1 до 99.9
- Два знака после запятой для удобства
- Случайный выбор между сложением и вычитанием
- Для вычитания гарантируется положительный результат

### Проверка ответов
- Учет погрешности в 0.01 для корректной проверки
- Подробные сообщения о правильности ответа
- Показ правильного ответа при ошибке
- Автоматическое отображение пошагового решения при неправильном ответе

### Пошаговое решение
- Детальное объяснение каждого шага вычисления
- Визуализация переноса разрядов при сложении
- Показ процесса заимствования при вычитании
- Интерактивная навигация по шагам решения
- Объяснение математических правил и алгоритмов

### Статистика
- Подсчет правильных ответов
- Общий счетчик заданий
- Процент успешности в реальном времени

### Режим тестирования
- 10 случайных вопросов с десятичными дробями
- Система оценок: 5 (90%+), 4 (75%+), 3 (60%+), 2 (40%+), 1 (<40%)
- Детальный анализ каждого ответа
- Возможность экспорта результатов
- Счетчик прогресса по вопросам
