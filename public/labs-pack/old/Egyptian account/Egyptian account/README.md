# 🏺 Египетская система счисления - Лабораторная работа

Интерактивная веб-страница для изучения древнеегипетской системы счисления, созданная для школьников.

## 📚 Описание

Этот проект представляет собой полноценную лабораторную работу по математике, которая помогает школьникам понять принципы работы египетской системы счисления через интерактивные упражнения и задания.

## ✨ Возможности

### 📖 Теория
- Подробное объяснение египетских иероглифов
- Примеры записи чисел
- Правила египетской системы счисления

### 🔄 Конвертер
- Перевод арабских чисел в египетские
- Подробное объяснение процесса конвертации
- Быстрые примеры для практики

### ✏️ Практика
- **Задание 1**: Интерактивный строитель египетских чисел
- **Задание 2**: Перевод египетских чисел в арабские
- Мгновенная обратная связь

### 🎯 Тест
- 5 вопросов для проверки знаний
- Подсчет результатов
- Визуальное отображение прогресса

## 🚀 Запуск

1. Скачайте все файлы проекта
2. Откройте файл `index.html` в любом современном браузере
3. Начните изучение с раздела "Теория"

## 🔧 Технические детали

### Египетские иероглифы
- 1: |
- 10: ∩
- 100: ⊂
- 1000: ⊃
- 10000: ⊄
- 100000: ⊅
- 1000000: ⊆

### Алгоритм конвертации
1. Разложение числа по разрядам
2. Замена каждого разряда соответствующими иероглифами
3. Группировка одинаковых символов
