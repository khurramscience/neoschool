# 🎈 Math Bubbles

An interactive web game for teaching students subtraction in a playful way.

## Game Description

"Math Bubbles" is an educational game where students learn subtraction by popping soap bubbles with math problems. The game helps develop quick calculation skills and concentration.

## How to Play

1. **Goal:** Pop only those bubbles where the subtraction result matches the target number
2. **Time:** You have 30 seconds for each level
3. **Levels:** Each level has a different target number (5, 3, 7, 4, 6)
4. **Scoring:**
   - +10 points for a correct bubble
   - -5 points for an incorrect bubble
   - Level is passed if you don't miss any correct bubbles and don't pop any wrong ones

## Game Features

- **Dynamic difficulty:** Problems get harder with each level
- **Beautiful animation:** Bubbles fall smoothly with a wobble
- **Visual feedback:** Correct bubbles are highlighted in green, incorrect ones in red
- **Responsive design:** Works on computers and mobile devices

## How to Start

1. Open `index.html` in any modern web browser
2. Click "Start Game"
3. Pop bubbles by clicking (or tapping on mobile)

## Technical Requirements

- Modern web browser with HTML5, CSS3, and JavaScript support
- Recommended: Chrome, Firefox, Safari, or Edge

## File Structure
