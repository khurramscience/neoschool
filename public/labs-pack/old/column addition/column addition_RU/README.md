# 🧮 Лабораторная работа: Сложение в столбик

Интерактивная веб-страница для обучения школьников сложению в столбик с пониманием разрядов чисел.

## 📚 Описание

Этот проект представляет собой полноценную лабораторную работу по математике, которая поможет ученикам:

- Понять, как работают разряды чисел
- Научиться складывать числа в столбик
- Закрепить знания через практические задания
- Проверить свои знания в игровой форме

## 🎯 Функциональность

### 📖 Обучение
- **Объяснение разрядов**: Визуальное представление сотен, десятков и единиц
- **Пошаговые примеры**: Детальное объяснение процесса сложения в столбик
- **Интерактивная демонстрация**: Анимированные примеры с возможностью управления

### 🎮 Практика
- **Случайные задания**: Генерация примеров для самостоятельного решения
- **Мгновенная проверка**: Проверка ответов с подробной обратной связью
- **Статистика**: Отслеживание прогресса и процента правильных ответов
- **Пошаговые решения**: Возможность посмотреть правильное решение

### 📝 Тест
- **10 вопросов**: Финальная проверка знаний
- **Прогресс-бар**: Визуальное отображение прогресса
- **Детальные результаты**: Анализ каждого ответа
- **Оценка**: Система оценок с мотивирующими сообщениями

## 🚀 Как использовать

1. Откройте файл `index.html` в любом современном браузере
2. Переходите между вкладками для изучения материала
3. Начните с вкладки "Обучение" для понимания теории
4. Перейдите к "Практике" для закрепления навыков
5. Завершите "Тестом" для проверки знаний
