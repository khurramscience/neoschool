# Division Laboratory - Interactive Math Learning

An interactive web-based laboratory for teaching division with and without remainder to elementary school students.

## Features

### 📚 Theory Section
- Visual explanations of division concepts
- Interactive examples with animated objects
- Clear definitions of mathematical terms (dividend, divisor, quotient, remainder)
- Step-by-step visual demonstrations

### 🎮 Practice Section
- Interactive drag-and-drop exercises
- Random problem generation with various object types
- Visual workspace for hands-on learning
- Real-time feedback and scoring
- Progress tracking

### ✅ Quiz Section
- 10-question knowledge assessment
- Multiple choice questions
- Immediate feedback on answers
- Score calculation and performance evaluation
- Motivational feedback based on results

## How to Use

1. **Open the Application**
   - Open `index.html` in any modern web browser
   - No additional setup required

2. **Navigate Between Sections**
   - Click on the navigation buttons at the top:
     - 📚 Theory - Learn the concepts
     - 🎮 Practice - Interactive exercises
     - ✅ Quiz - Test your knowledge

3. **Theory Section**
   - Read through the explanations
   - Observe the visual examples
   - Hover over objects to see interactive effects

4. **Practice Section**
   - Click "New Problem" to generate a random division problem
   - Drag objects from the top area into group slots below
   - Enter your answers for quotient and remainder
   - Click "Check Answer" to verify your solution
   - Use "Reset" to try the same problem again

5. **Quiz Section**
   - Click "Start Quiz" to begin
   - Answer 10 multiple-choice questions
   - See immediate feedback for each answer
   - Review your final score and performance
