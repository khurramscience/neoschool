# 🧮 Математическая Память - Лабораторная работа

Интерактивная веб-игра для развития памяти школьников с математическими примерами.

## 📋 Описание

Это веб-приложение представляет собой игру "Память" с математическим уклоном. Ученики должны найти пары карточек: математический пример и его правильный ответ. Игра помогает развивать память, внимание и математические навыки.

## 🎮 Функциональность

### Основные возможности:
- **Два режима игры:**
  - **Режим подсчёта шагов** - считает количество попыток для нахождения всех пар
  - **Режим подсчёта времени** - засекает время, затраченное на игру

- **Три уровня сложности:**
  - **Легкая** - 6 пар карточек (4×3 сетка)
  - **Средняя** - 8 пар карточек (4×4 сетка) 
  - **Сложная** - 12 пар карточек (6×4 сетка)

- **Математические операции:**
  - Сложение (+)
  - Вычитание (-)
  - Умножение (×)

### Игровой процесс:
1. Ученик выбирает режим игры и сложность
2. Нажимает "Начать игру"
3. Карточки появляются рубашкой вверх
4. Ученик кликает по карточке, чтобы увидеть пример или число
5. Затем выбирает вторую карточку
6. Если пример и ответ совпадают - пара найдена
7. Игра продолжается до нахождения всех пар

## 🚀 Запуск

1. Откройте файл `index.html` в любом современном браузере
2. Или запустите локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (если установлен)
   npx http-server
   ```
