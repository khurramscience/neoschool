Adding and Subtracting Negative Numbers

Basic Rules for Working with Negative Numbers

🔢 Adding Negative Numbers

When adding two negative numbers:

  • Add their absolute values (moduli)
  • Put a minus sign before the result
Example: (-5) + (-3) = -(5 + 3) = -8

➖ Subtracting Negative Numbers

When subtracting a negative number:

  • Subtracting a negative number = adding a positive number
  • a - (-b) = a + b
Example: 7 - (-4) = 7 + 4 = 11

➕ Adding Numbers with Different Signs

When adding a positive and negative number:

  • Subtract the smaller absolute value from the larger one
  • Use the sign of the number with the larger absolute value
Example: 8 + (-3) = 8 - 3 = 5

🔢 Working with Fractions

When adding and subtracting fractions with negative numbers:

  • Find a common denominator
  • Apply the same sign rules
  • Negative fraction: -a/b = (-a)/b = a/(-b)
Example: 1/2 + (-1/3) = 3/6 + (-2/6) = 1/6
