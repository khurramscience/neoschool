# 🧮 Laboratory Work: Adding and Subtracting Negative Numbers

Interactive web page for learning addition and subtraction of negative numbers, including work with fractions.

## 📚 Description

This project is a full-featured mathematics laboratory work that will help students master:

- Adding negative numbers
- Subtracting negative numbers  
- Adding numbers with different signs
- Working with fractions and negative numbers

## 🎯 Functionality

### 1. 📚 Theory
- Detailed explanations of rules for working with negative numbers
- Examples for each rule
- Separate section on working with fractions
- Interactive cards with animations

### 2. ✏️ Practice
- Generation of random tasks of different types
- Real-time answer checking
- Display of detailed solutions
- Statistics of correct answers
- Support for fraction input (e.g., 1/2, -3/4)

### 3. 📝 Knowledge Check
- Test with 10 questions
- Execution timer
- Input answers in number or fraction format
- Support for common fractions (e.g., 1/2, -3/4)
- Detailed analysis of results
- Possibility to retake

## 🚀 Launch

1. Download all project files
2. Open the `index.html` file in any modern browser
3. Start learning from the "Theory" section

## 🎓 Task Types

### Integers
- Adding two negative numbers: `(-5) + (-3)`
- Subtracting a negative number: `7 - (-4)`
- Adding numbers with different signs: `8 + (-3)`

### Fractions
- Adding fractions: `1/2 + (-1/3)`
- Subtracting fractions: `3/4 - (-1/2)`
- Mixed operations with fractions
- Support for entering answers in fraction format: `1/2`, `-3/4`

## 🔢 Fraction Answer Support

The system supports entering answers in various formats:
- **Integers**: `5`, `-3`, `0`
- **Decimal fractions**: `2.5`, `-1.75`
- **Common fractions**: `1/2`, `-3/4`, `2/3`
- **Mixed numbers**: `1 1/2` (with space)

**Examples of correct input:**
- `1/2` for answer 0.5
- `-3/4` for answer -0.75
- `5` for integer
- `2.5` for decimal fraction
