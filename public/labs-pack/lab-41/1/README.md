# 🧮 Math Laboratory

Interactive web page for training mathematical skills of school students. The application helps develop addition, subtraction and multiplication skills in a game format.

## ✨ Features

- **Three mathematical operations**: addition (+), subtraction (-), multiplication (×)
- **Customizable time**: from 10 to 300 seconds
- **Three difficulty levels**:
  - Easy: numbers from 1 to 10
  - Medium: numbers from 1 to 50
  - Hard: numbers from 1 to 100
- **Results statistics**: number of correct answers, success rate
- **Error analysis**: showing all wrong answers with correct solutions
- **Modern design**: beautiful and intuitive interface
- **Responsive**: works on computers, tablets and phones

## 🚀 How to use

1. Open the `index.html` file in any modern browser
2. Configure game settings:
   - Select time (default 60 seconds)
   - Check needed operations (addition, subtraction, multiplication)
   - Choose difficulty level
3. Click "Start Game"
4. Solve examples by entering answers in the input field
5. After time runs out or clicking "End Game" view the results
6. Study your mistakes and try again!

## 🎮 Game features

- **Timer**: countdown with warning in the last 10 seconds
- **Instant feedback**: visual hints for correct and wrong answers
- **Auto-generated questions**: new examples appear automatically after each answer
- **Score tracking**: real-time tracking of correct answers
- **Detailed analysis**: complete list of mistakes with correct answers
