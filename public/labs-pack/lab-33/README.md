# 🧮 Математическая лаборатория - Изучаем деление

Интерактивная веб-страница для изучения операции деления с помощью визуализации групп объектов. Создана специально для школьников, чтобы помочь им понять принципы деления через наглядные примеры.

## ✨ Особенности

- **Два режима работы**: Автоматический и интерактивный
- **Автоматический режим**: Показывает готовые примеры деления с визуализацией
- **Интерактивный режим**: Позволяет ученикам самим создавать группы, перетаскивая объекты
- **Мгновенные вычисления**: Результаты обновляются в реальном времени
- **Понятные объяснения**: Каждое действие сопровождается текстовым объяснением
- **Современный дизайн**: Красивый и отзывчивый интерфейс с градиентами и анимациями
- **Адаптивность**: Работает на всех устройствах - компьютерах, планшетах и телефонах

## 🎯 Как использовать

### Автоматический режим
1. **Переключитесь на "Автоматический режим"**
2. **Введите параметры**:
   - Общее количество объектов (от 1 до 50)
   - Размер группы (от 1 до 20)
3. **Нажмите "Рассчитать деление"** или просто измените значения в полях
4. **Наблюдайте результат**:
   - Математическое выражение
   - Визуализация групп объектов
   - Количество групп и остаток
   - Понятное объяснение

### Интерактивный режим
1. **Переключитесь на "Интерактивный режим"**
2. **Введите количество объектов** (от 1 до 30) и нажмите "Создать объекты"
3. **Установите количество групп** (от 1 до 15) и нажмите "Обновить группы"
4. **Перетаскивайте объекты** из области "Свободные объекты" в область "Созданные группы"
5. **Создавайте группы** любого размера, перетаскивая объекты в пустые слоты
6. **Нажмите "Проверить результат"** для получения подробной статистики и подсказок
7. **Используйте "Сбросить все"** для начала заново

## 📊 Панель состояния (интерактивный режим)

В интерактивном режиме отображается панель с информацией:
- **Всего объектов**: Общее количество созданных объектов
- **В группах**: Сколько объектов уже распределено по группам
- **Свободных**: Сколько объектов осталось нераспределенными
- **Создано групп**: Сколько групп создано в данный момент
- **Целевых групп**: Заданное количество групп для создания

## 📈 Подробная статистика и анализ

При нажатии кнопки "Проверить результат" отображается подробный анализ:

### 📊 Основная статистика:
- Общее количество объектов
- Количество созданных групп
- Целевое количество групп
- Количество свободных объектов

### 🔍 Анализ групп:
- Размер каждой группы с цветовой индикацией
- Максимальный и минимальный размер групп
- Средний размер группы
- Разница в размерах между группами

### 💡 Подсказки:
- Соответствие заданному количеству групп
- Анализ равномерности распределения
- Ожидаемый размер групп
- Информация об остатке при делении

### 🎯 Рекомендации:
- Конкретные действия для улучшения результата
- Советы по созданию равных групп
- Направления для дальнейшей работы
- Поощрение за правильное выполнение

## 📚 Образовательные возможности

### Что изучают школьники:
- **Понятие деления** как распределения объектов по группам
- **Целочисленное деление** и остаток
- **Визуальная связь** между числами и группами объектов
- **Практическое применение** математических операций
- **Интерактивное обучение** через перетаскивание объектов
- **Анализ результатов** собственных действий

### Примеры для изучения:
- `12 ÷ 3 = 4` - 12 объектов делим на группы по 3, получаем 4 группы
- `17 ÷ 5 = 3 (остаток 2)` - 17 объектов делим на группы по 5, получаем 3 полные группы и 2 объекта останется
- `8 ÷ 2 = 4` - 8 объектов делим на группы по 2, получаем 4 группы

### Интерактивные возможности:
- **Создание равных групп**: Ученики учатся делить объекты поровну
- **Понимание остатка**: Когда деление не выполняется нацело
- **Гибкость группировки**: Можно создавать группы разного размера
- **Визуальная обратная связь**: Мгновенное отображение результатов
- **Управление количеством групп**: Возможность задать точное количество групп для создания
- **Динамическое изменение**: Добавление и удаление групп в процессе работы
- **Целевое обучение**: Создание конкретного количества групп для решения задач
- **Подробная статистика**: Детальный анализ созданных групп с подсказками
- **Умные рекомендации**: Персонализированные советы для улучшения результата
- **Цветовая индикация**: Визуальное отображение качества группировки
