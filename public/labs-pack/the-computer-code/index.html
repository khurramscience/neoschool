📚 What is the binary system?

The binary number system is a system that uses only two digits: 0 and 1.

Each position in a binary number represents a power of 2:

1 0 1 1
2³ 2² 2¹ 2⁰

1011₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 11₁₀
