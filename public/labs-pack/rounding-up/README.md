# 🔢 Lab Work: Rounding Numbers

Interactive web page for learning number rounding rules for school students.

## 📋 Description

This web application is designed for learning and practicing number rounding. It includes:

- **Theoretical section** with explanation of rounding rules
- **Interactive practice** for independent number rounding
- **Exercise system** with automatic answer checking
- **Statistics** for tracking progress

## 🎯 Features

### 📚 Theory
- Detailed explanation of rounding rules
- Visual examples with explanations
- Visual representation of the algorithm

### 🎯 Practice
- Input any number for rounding
- Choose rounding precision (to whole numbers, 1, 2, 3 decimal places)
- Instant result with explanation
- Detailed explanation of each rounding step

### 📝 Exercises
- Automatic generation of random tasks
- Answer correctness checking
- Immediate feedback
- Various difficulty levels

### 📊 Statistics
- Count of total exercises
- Tracking of correct answers
- Accuracy percentage calculation
- Saving results in browser

## 🚀 How to Use

1. **Open the `index.html` file** in any modern browser
2. **Study the theory** in the "Theory" section
3. **Try the practice** - enter a number and choose rounding precision
4. **Solve exercises** - click "Create New Exercise"
5. **Track your progress** in the "Statistics" section

## 📖 Rounding Rules

### Main Rule:
- If the digit after the place we're rounding to is **5 or greater**, then we increase the digit in that place by 1
- If the digit is **less than 5**, then we leave the digit unchanged

### Examples:
- 3.7 → 4 (7 > 5, round up)
- 3.4 → 3 (4 < 5, round down)
- 3.5 → 4 (5 = 5, round up)
