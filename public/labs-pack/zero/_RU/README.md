# 🔢 Lab Work: The Importance of Zero in Mathematics

Interactive web page for schoolchildren that helps understand the importance of zero when counting objects.

## 📋 Description

This lab work consists of three main sections:

### 📚 Theory
- Explanation of what zero is
- Examples of using zero in everyday life
- Demonstration of the importance of zero in the positional number system
- Visual examples with emojis

### 🎯 Practice
**Exercise 1: Counting Objects**
- Interactive game with adding/removing objects
- Students can add fruits and count their quantity
- Verification of counting accuracy

**Exercise 2: Fill in the Blanks**
- Tasks for filling in blanks with numbers
- Includes cases with zero
- Instant answer checking

### ✅ Knowledge Check
- Interactive test with 5 questions
- Questions about understanding the importance of zero
- Detailed results with explanations
- Ability to retake the test

## 🚀 How to Run

1. Download all files to one folder:
   - `index.html`
   - `styles.css`
   - `script.js`

2. Open the `index.html` file in any modern browser

3. Start learning with the "Theory" section

## 🎯 Learning Objectives

After completing the lab work, students should:

1. **Understand** the meaning of zero as the absence of quantity
2. **Realize** the importance of zero in mathematics
3. **Be able** to correctly use zero when counting
4. **Understand** the role of zero in the positional number system
