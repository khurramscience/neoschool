# 🔢 Lab Work: Ternary Number System

Interactive web page for learning the ternary number system, designed for school students.

## 📋 Description

This lab work is a comprehensive web application for studying the ternary number system. The application includes theoretical material, interactive tools, and practical exercises.

## 🎯 Learning Objectives

- Understand the principles of the ternary number system
- Learn to convert numbers between decimal and ternary systems
- Master arithmetic operations in the ternary system
- Reinforce knowledge through practical exercises

## 🚀 Features

### 📚 Theory
- Detailed explanation of ternary system principles
- Step-by-step conversion examples
- Comparison table of number systems
- Visual calculation examples

### 🔄 Converter
- Conversion from decimal to ternary system
- Conversion from ternary to decimal system
- Step-by-step display of conversion process
- Input validation

### 🧮 Calculator
- Arithmetic operations in ternary system
- Automatic conversion between systems
- Detailed display of calculations
- Keyboard input support

### 🎯 Practice
- Various types of exercises
- Automatic answer checking
- Hint system
- Progress tracking
