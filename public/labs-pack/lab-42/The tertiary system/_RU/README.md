# 🔢 Лабораторная работа: Третичная система счисления

Интерактивная веб-страница для изучения третичной системы счисления, предназначенная для школьников.

## 📋 Описание

Данная лабораторная работа представляет собой полноценное веб-приложение для изучения третичной системы счисления. Приложение включает в себя теоретический материал, интерактивные инструменты и практические задания.

## 🎯 Цели обучения

- Понять принципы работы третичной системы счисления
- Научиться переводить числа между десятичной и третичной системами
- Освоить арифметические операции в третичной системе
- Закрепить знания через практические задания

## 🚀 Возможности

### 📚 Теория
- Подробное объяснение принципов третичной системы
- Пошаговые примеры конвертации
- Сравнительная таблица систем счисления
- Визуальные примеры вычислений

### 🔄 Конвертер
- Перевод из десятичной в третичную систему
- Перевод из третичной в десятичную систему
- Пошаговое отображение процесса конвертации
- Валидация ввода

### 🧮 Калькулятор
- Арифметические операции в третичной системе
- Автоматический перевод между системами
- Подробное отображение вычислений
- Поддержка клавиатурного ввода

### 🎯 Практика
- Разнообразные типы заданий
- Автоматическая проверка ответов
- Система подсказок
- Отслеживание прогресса
