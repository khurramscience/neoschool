📚 Что такое третичная система счисления?

Третичная система счисления — это позиционная система счисления с основанием 3. В ней используются только три цифры: 0, 1 и 2.

Основные принципы:

  • Каждая позиция представляет степень числа 3
  • Цифры: 0, 1, 2
  • Позиции справа налево: 3⁰, 3¹, 3², 3³, ...

Пример перевода из десятичной в третичную:

Число 25 в десятичной системе:

25 ÷ 3 = 8 остаток 1

8 ÷ 3 = 2 остаток 2

2 ÷ 3 = 0 остаток 2

Результат: 221₃
