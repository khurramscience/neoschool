# 🎯 Математическая Игра - Сумма Последовательностей

Интерактивная веб-игра для обучения школьников математике в игровой форме. Игра помогает развивать навыки счета, логическое мышление и скорость реакции.

## 🎮 Как играть

1. **Цель игры**: Найдите последовательности чисел, сумма которых равна заданной целевой сумме
2. **Управление**: Кликайте на числа и ведите курсор по горизонтали или вертикали (не по диагонали)
3. **Подсчет очков**: Чем длиннее последовательность, тем больше очков вы получаете
4. **Время**: У вас есть 60 секунд, чтобы набрать как можно больше очков
5. **Уровни**: С каждым новым уровнем целевая сумма увеличивается

## 🚀 Запуск игры

1. Откройте файл `index.html` в любом современном веб-браузере
2. Нажмите кнопку "Начать Игру"
3. Наслаждайтесь игрой!

## 🎨 Кастомизация

Вы можете легко изменить параметры игры, отредактировав переменные в файле `script.js`:

```javascript
this.boardSize = 8;        // Размер игрового поля
this.targetSum = 15;       // Начальная целевая сумма
this.timeLeft = 60;        // Время игры в секундах
```
