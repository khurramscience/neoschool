# 🔢 Learn the Binary Number System

An interactive web page for students to help understand the principles of the binary number system.

## 📚 Description

This project is an educational web page that helps students learn the binary number system through interactive elements and practical exercises.

## ✨ Features

### 📖 Theory Section
- Explanation of the principles of the binary number system
- Visual representation of digit positions and their weights
- Conversion examples

### 🔄 Interactive Converter
- Convert decimal numbers to binary (0-255)
- Beautiful 8-bit representation
- Instant result

### 🎮 Interactive Bits
- Clickable bits for exploration
- Visual feedback
- Instant calculation of decimal value

### 📝 Exercises
- Random conversion tasks
- Answer checking with explanations
- Generate new tasks

### 📊 Powers of Two Table
- Reference table of powers from 2⁰ to 2¹⁰
- Helps understand digit weights

## 🚀 How to Use

1. Open the `index.html` file in any modern browser
2. Study the theory section
3. Try the number converter
4. Play with the interactive bits
5. Solve the exercises
6. Use the powers of two table as a reference

## 🎯 Learning Goals

After working with this app, students should:

- ✅ Understand the principles of the binary number system
- ✅ Be able to convert numbers from decimal to binary
- ✅ Understand the weights of digits in the binary system
- ✅ Be able to read and write binary numbers
- ✅ Solve simple conversion tasks
