# 🏛️ Лабораторная работа: Римская система счета

Интерактивная веб-страница для изучения римской системы счета, созданная для школьников.

## 📚 Описание

Эта лабораторная работа поможет учащимся понять и освоить римскую систему счета через интерактивные упражнения, конвертеры и игры.

## 🎯 Возможности

### 📖 Теория
- Подробное объяснение римских цифр
- Таблица соответствий римских и арабских чисел
- Правила записи римских чисел
- Примеры использования

### 🔄 Конвертер
- Перевод арабских чисел в римские (1-3999)
- Перевод римских чисел в арабские
- Быстрый конвертер для популярных чисел
- Проверка корректности ввода

### ✏️ Упражнения
- Генерация случайных заданий
- Различные типы упражнений:
  - Перевод арабских чисел в римские
  - Перевод римских чисел в арабские
  - Сложение римских чисел
  - Вычитание римских чисел
- Подсказки для сложных заданий
- Статистика правильных ответов

### 🎮 Игра
- Интерактивная игра на время
- 60 секунд на решение заданий
- 4 варианта ответа для каждого вопроса
- Подсчет очков
- Различные типы вопросов

## 🚀 Запуск

1. Скачайте все файлы в одну папку
2. Откройте файл `index.html` в любом современном браузере
3. Начните изучение!
