# 🏛️ Lab Work: Roman Numeral System

Interactive web page for learning the Roman numeral system, created for school students.

## 📚 Description

This lab work will help students understand and master the Roman numeral system through interactive exercises, converters, and games.

## 🎯 Features

### 📖 Theory
- Detailed explanation of Roman numerals
- Table of correspondences between Roman and Arabic numbers
- Rules for writing Roman numerals
- Usage examples

### 🔄 Converter
- Convert Arabic numbers to Roman (1-3999)
- Convert Roman numbers to Arabic
- Quick converter for popular numbers
- Input validation

### ✏️ Exercises
- Random exercise generation
- Various types of exercises:
  - Convert Arabic numbers to Roman
  - Convert Roman numbers to Arabic
  - Add Roman numbers
  - Subtract Roman numbers
- Hints for difficult tasks
- Statistics of correct answers

### 🎮 Game
- Interactive timed game
- 60 seconds to solve tasks
- 4 answer options for each question
- Score tracking
- Various question types

## 🚀 Launch

1. Download all files to one folder
2. Open the `index.html` file in any modern browser
3. Start learning!
