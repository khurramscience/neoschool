📚 Basics of Decimal Fractions

What is a Decimal Fraction?

A decimal fraction is a way to write an ordinary fraction where the denominator is 10, 100, 1000, etc.

Examples:
  • 0.5 = 5/10 = 1/2
  • 0.25 = 25/100 = 1/4
  • 0.125 = 125/1000 = 1/8

hundreds tens ones tenths hundredths thousandths
