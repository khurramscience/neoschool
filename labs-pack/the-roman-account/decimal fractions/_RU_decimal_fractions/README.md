# 🔢 Лабораторная работа: Десятичные дроби

Интерактивная веб-страница для изучения десятичных дробей школьниками.

## 📚 Описание

Эта лабораторная работа поможет ученикам понять концепцию десятичных дробей через:
- **Теоретический материал** с примерами
- **Интерактивную визуализацию** дробей
- **Практические задания** для закрепления
- **Обучающую игру** для проверки знаний

## 🎯 Возможности

### 📖 Теория
- Объяснение понятия десятичных дробей
- Примеры конвертации обыкновенных дробей в десятичные
- Визуальное представление разрядов десятичных чисел

### 🎨 Визуализация
- Интерактивная сетка 10x10 для наглядного представления дробей
- Ввод любой десятичной дроби от 0 до 1
- Автоматическое отображение в виде обыкновенной дроби и процентов
- Цветовое выделение заполненных ячеек

### ✏️ Практика
**Задание 1: Конвертация дробей**
- Перевод обыкновенных дробей в десятичные
- Мгновенная проверка ответов
- Подробная обратная связь

**Задание 2: Сравнение дробей**
- Сравнение двух десятичных дробей
- Выбор правильного знака сравнения (<, =, >)
- Объяснение правильного ответа

### 🎮 Игра
- Поиск правильной десятичной дроби среди вариантов
- Система уровней сложности
- Подсчет очков
- Автоматическое повышение уровня при успешном прохождении

## 🚀 Запуск

1. Скачайте все файлы в одну папку
2. Откройте файл `index.html` в любом современном браузере
3. Начните изучение с раздела "Теория"
