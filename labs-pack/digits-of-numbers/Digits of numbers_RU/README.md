# 🔢 Изучаем разряды чисел

Интерактивная веб-страница для изучения разрядов чисел, созданная для школьников.

## 📚 Описание

Этот проект представляет собой образовательную веб-страницу, которая помогает школьникам понять, как работают разряды чисел. Страница включает в себя:

- **Интерактивный разбор чисел** - введите любое число и увидите его разбор по разрядам
- **Практические задания** - генерируйте случайные числа для тренировки
- **Теоретический материал** - объяснение понятия разрядов с примерами
- **Современный дизайн** - красивый и удобный интерфейс

## 🚀 Как использовать

1. Откройте файл `index.html` в любом современном браузере
2. Введите число в поле ввода и нажмите "Разобрать число"
3. Изучите результат разбора по разрядам
4. Попробуйте практические задания, нажав "Сгенерировать число"
5. Изучите теоретический материал внизу страницы

## ✨ Функции

### 📊 Разбор чисел
- Введите любое положительное число
- Получите визуальный разбор по разрядам
- Увидите значение каждой цифры в зависимости от её позиции

### 🎯 Практические задания
- Генерация случайных чисел для тренировки

### 📖 Теория

- Подробный пример разбора числа
