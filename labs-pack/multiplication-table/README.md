# Multiplication Table Laboratory

An interactive web-based learning tool designed to help students master multiplication tables through theory, practice, and testing.

## Features

### 🎯 Three Learning Sections

1. **Theory Section**
   - Comprehensive explanation of multiplication tables
   - Interactive table with hover effects
   - Visual highlighting of rows and columns

2. **Practice Section**
   - Click on any cell to see the multiplication problem
   - Visual feedback with highlighted cells
   - Reset functionality for continuous practice

3. **Test Section**
   - 10 random multiplication questions
   - Real-time scoring and feedback
   - Reference table available during testing

### 🎨 Interactive Features

- **Hover Effects**: Move your cursor over the table to see row and column highlighting
- **Visual Feedback**: Different colors for different interactions
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient background and smooth animations

### 📱 Responsive Design

- Optimized for all screen sizes
- Touch-friendly interface for mobile devices
- Adaptive layout for different devices

## How to Use

1. **Open the Application**
   - Simply open `index.html` in any modern web browser
   - No installation or setup required

2. **Navigate Between Sections**
   - Use the navigation buttons at the top
   - Each section provides different learning experiences

3. **Theory Section**
   - Read the explanations about multiplication tables
   - Hover over the table to see interactive highlighting
   - Understand how to read and use multiplication tables

4. **Practice Section**
   - Click on any cell in the table
   - See the multiplication problem and its solution
   - Use the reset button to start over

5. **Test Section**
   - Click "Start Test" to begin
   - Answer 10 random multiplication questions
   - Get immediate feedback on your answers
   - View your final score and performance

## Customization

The application can be easily customized:

- **Table Size**: Modify the `size` variable in `script.js` (currently 12x12)
- **Colors**: Update CSS variables for different color schemes
- **Questions**: Adjust the number of test questions in the `generateTestQuestions()` function
- **Content**: Modify text content in `index.html` for different languages or explanations
