📚 Theoretical Part

What is Zero?

Zero (0) is a number that means the absence of quantity. This is a very important number in mathematics!

Examples of using zero:

  • You have 0 apples = you have no apples
  • There are 0 candies in the box = the box is empty
  • There are 0 books on the table = the table is empty

Why is Zero So Important?

Without zero, we couldn't accurately say that we have nothing. Imagine you're counting candies:

1 🍬
2 🍬🍬
0 (empty)

Without zero, we wouldn't know that there are no more candies!

Zero in Positional System

Zero helps us write large numbers correctly:

10 one ten, zero ones
100 one ten of tens, zero tens, zero ones
