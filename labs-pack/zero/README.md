# 🔢 Лабораторная работа: Важность нуля в математике

Интерактивная веб-страница для школьников, которая помогает понять важность нуля при подсчете предметов.

## 📋 Описание

Эта лабораторная работа состоит из трех основных блоков:

### 📚 Теория
- Объяснение, что такое ноль
- Примеры использования нуля в повседневной жизни
- Демонстрация важности нуля в позиционной системе счисления
- Визуальные примеры с эмодзи

### 🎯 Практика
**Упражнение 1: Подсчет предметов**
- Интерактивная игра с добавлением/удалением предметов
- Ученики могут добавлять фрукты и считать их количество
- Проверка правильности подсчета

**Упражнение 2: Заполни пропуски**
- Задания на заполнение пропусков числами
- Включает случаи с нулем
- Мгновенная проверка ответов

### ✅ Проверка знаний
- Интерактивный тест из 5 вопросов
- Вопросы о понимании важности нуля
- Подробные результаты с объяснениями
- Возможность пройти тест заново

## 🚀 Как запустить

1. Скачайте все файлы в одну папку:
   - `index.html`
   - `styles.css`
   - `script.js`

2. Откройте файл `index.html` в любом современном браузере

3. Начните изучение с раздела "Теория"

## 🎯 Цели обучения

После прохождения лабораторной работы ученики должны:

1. **Понимать** значение нуля как отсутствие количества
2. **Осознавать** важность нуля в математике
3. **Уметь** правильно использовать ноль при подсчете
4. **Понимать** роль нуля в позиционной системе счисления
