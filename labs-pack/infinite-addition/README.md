# 🧮 Laboratory Work: Adding Three Numbers in Column

Interactive web page for teaching students to add three numbers in column with digit carryover visualization.

## 🎯 Project Goal

Help students understand and master:
- Adding three numbers in column
- Digit carryover mechanism in addition
- Step-by-step algorithm for performing addition
- Visual representation of calculation process

## ✨ Main Features

### 📊 Difficulty Adjustment
- Choice of number of digits (2-5 digits)
- Automatic problem generation
- Gradual increase in difficulty

### 👁️ Carryover Visualization
- Interactive display of carried numbers
- Ability to show/hide carryovers
- Color highlighting of carried digits

### ✅ Verification and Learning
- Interactive answer input by digits
- Instant correctness verification
- Detailed step-by-step solution explanation
- Performance statistics

### 📈 Statistics
- Number of solved problems
- Percentage of correct answers
- Learning progress tracking

## 🚀 How to Use

1. **Open the `index.html` file in a web browser**
2. **Select the number of digits** from the dropdown list
3. **Click "Generate new problem"** to create a task
4. **Solve the problem** by entering digits in the answer fields
5. **Click "Check answer"** to verify
6. **Use "Show solution"** for detailed explanation
7. **Click "👁️ Show carryovers"** to visualize carried numbers

## 🎓 Educational Features

### Step-by-step Learning
- Start with simple problems (2 digits)
- Gradually move to more complex ones (3-5 digits)
- Use the solution display function to understand the algorithm

### Visual Support
- Carried numbers are highlighted in red
- Step-by-step explanation with position names
- Interactive elements for better understanding

### Adaptive Learning
- Statistics help track progress
- Ability to re-solve problems
- Instant feedback
