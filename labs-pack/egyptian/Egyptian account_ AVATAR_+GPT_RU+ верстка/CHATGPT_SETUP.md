# 🤖 Настройка ChatGPT для чата

## 📋 Что нужно сделать

### 1. Получение API ключа OpenAI
1. Перейдите на сайт [OpenAI](https://platform.openai.com/)
2. Войдите в свой аккаунт или создайте новый
3. Перейдите в раздел "API Keys"
4. Нажмите "Create new secret key"
5. Скопируйте полученный ключ (он начинается с `sk-`)

### 2. Настройка в приложении
1. Откройте приложение в браузере
2. В чате появится поле для ввода API ключа
3. Вставьте ваш API ключ в поле
4. Нажмите кнопку 💾 для сохранения
5. Статус изменится на "🟢 API настроен"

### 3. Использование
- После настройки API чат будет использовать ChatGPT
- Задавайте любые вопросы о египетской системе счисления
- ChatGPT будет отвечать на русском языке
- Иероглифы будут отображаться как картинки

### 4. Режимы работы
- **🔄 Автоматический**: Использует ChatGPT если API настроен, иначе локальные ответы
- **💻 Локальный**: Всегда использует встроенные ответы (быстро и бесплатно)
- **🤖 ChatGPT**: Всегда использует ChatGPT API (требует API ключ)

Переключайте режимы кнопкой в заголовке чата!

## ⚠️ Важные моменты

### Безопасность
- API ключ сохраняется только в вашем браузере
- Никогда не делитесь своим API ключом
- Ключ можно скрыть/показать кнопкой 👁️

### Стоимость
- Использование ChatGPT API платное
- Стоимость зависит от количества токенов
- Примерно $0.002 за 1000 токенов
- Для обучения достаточно небольшого бюджета

### Ограничения
- API ключ действителен только для вашего аккаунта
- Есть лимиты на количество запросов
- При превышении лимитов запросы могут не проходить

## 🔧 Устранение проблем

### API ключ не работает
- Проверьте правильность ключа
- Убедитесь, что у вас есть средства на счете
- Проверьте лимиты API

### Ошибки сети
- Проверьте интернет-соединение
- Попробуйте обновить страницу
- Убедитесь, что сайт OpenAI доступен

### Ошибка 429 (Too Many Requests)
- **Причина**: Превышен лимит запросов к API
- **Решение**: 
  - Подождите несколько минут
  - Проверьте баланс на [platform.openai.com](https://platform.openai.com/usage)
  - Переключитесь на локальный режим
  - Используйте кнопку "Локальный режим" в чате

### Медленные ответы
- Это нормально для ChatGPT
- Ответы могут занимать 5-15 секунд
- При высокой нагрузке время может увеличиться
