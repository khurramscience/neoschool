# 🔢 Лабораторная работа: Переместительное свойство умножения

Интерактивная веб-страница для изучения переместительного свойства умножения (коммутативности) в формате лабораторной работы для школьников.

## 📚 Описание

Данная лабораторная работа помогает ученикам понять и закрепить переместительное свойство умножения, которое гласит, что **a × b = b × a**. Проект состоит из трех основных блоков:

### 1. 📖 Теория
- Объяснение переместительного свойства умножения
- Математическая формула: a × b = b × a
- Наглядные примеры с вычислениями
- Визуальное представление с помощью точек

### 2. ✏️ Практика
- **Задание 1**: Интерактивный калькулятор для проверки свойства
- **Задание 2**: Создание собственных примеров с автоматической проверкой

### 3. 🧪 Проверка знаний
- Тест из 10 вопросов
- Автоматическая проверка правильности
- Детальные результаты с объяснениями
- Возможность повторного прохождения

## 🚀 Как запустить

1. Скачайте все файлы проекта
2. Откройте файл `index.html` в любом современном браузере
3. Начните изучение с раздела "Теория"
