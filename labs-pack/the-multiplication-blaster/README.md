# 🎯 Multiplication Blaster

Interactive web game for learning multiplication with a paintball gun!

## 🎮 Game Description

Multiplication Blaster is an exciting game that helps children understand the concept of multiplication through visual representation. The player uses a paintball gun to create matrices of colored balls and solve mathematical problems.

## 🎯 How to Play

1. **Choose line length** (from 1 to 9) using the slider
2. **Click "Shoot!"** to create a line of balls on the game field
3. **Repeat several times** to create a matrix
4. **Achieve the correct result** of multiplication!

## 🎨 Features

- **Visual Learning**: Children see how multiplication works in practice
- **Interactive Interface**: Modern design with animations
- **5 Difficulty Levels**: From simple to complex examples
- **Scoring System**: Motivation for achieving better results
- **Responsive Design**: Works on all devices

## 🚀 Running the Game

1. Download all files to one folder
2. Open the `index.html` file in any modern browser
3. Enjoy the game!

## 🎓 Educational Value

The game helps children:
- Understand the concept of multiplication as repeated addition
- Visualize mathematical operations
- Develop spatial thinking
- Improve problem-solving skills

## 🎯 Game Levels

1. **Level 1**: 3 × 4 = 12
2. **Level 2**: 5 × 3 = 15
3. **Level 3**: 6 × 7 = 42
4. **Level 4**: 4 × 8 = 32
5. **Level 5**: 7 × 9 = 63
