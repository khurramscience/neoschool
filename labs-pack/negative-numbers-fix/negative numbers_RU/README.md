# 🔢 Лабораторная работа: Отрицательные числа

Интерактивная веб-страница для изучения отрицательных чисел школьниками с помощью реальных примеров из жизни.

## 📋 Описание

Эта лабораторная работа поможет ученикам понять концепцию отрицательных чисел через простые и понятные примеры из повседневной жизни:

- **🌡️ Температура** - понимание отрицательной температуры
- **💰 Деньги** - понятие долга и отрицательного баланса
- **🏢 Этажи здания** - подземные этажи как отрицательные числа

## 🎯 Структура

### 1. 📚 Теория
- Объяснение что такое отрицательные числа
- Интерактивные примеры из реальной жизни
- Визуальные демонстрации

### 2. 🎯 Тренировка
- **Задание 1**: Работа с температурой (слайдер + кнопка "потеплеть")
- **Задание 2**: Калькулятор денег (покупки и долги)
- **Задание 3**: Калькулятор этажей (подъём и спуск)

### 3. 📝 Проверка знаний
- 5 вопросов с вариантами ответов
- Автоматическая проверка
- Детальный разбор результатов
- Возможность пройти тест заново

## 🚀 Как запустить

1. Скачайте все файлы в одну папку
2. Откройте файл `index.html` в любом современном браузере
3. Начните изучение с раздела "Теория"

## 📁 Файлы проекта

- Применить знания в реальных ситуациях

## 💡 Примеры использования

### Температура
- Утром -5°C, днём потеплело на 8°C → стало +3°C

### Деньги
- Есть 100 рублей, купил игрушку за 150 рублей → долг 50 рублей (-50)

### Этажи
- Находишься на 2-м этаже, спускаешься на 5 этажей → оказываешься на -3-м этаже (подвал)
