# 🔢 Laboratory Work: Negative Numbers

Interactive web page for studying negative numbers by schoolchildren using real-life examples.

## 📋 Description

This laboratory work will help students understand the concept of negative numbers through simple and understandable examples from everyday life:

- **🌡️ Temperature** - understanding negative temperature
- **💰 Money** - concept of debt and negative balance
- **🏢 Building floors** - underground floors as negative numbers

## 🎯 Structure

### 1. 📚 Theory
- Explanation of what negative numbers are
- Interactive examples from real life
- Visual demonstrations

### 2. 🎯 Practice
- **Exercise 1**: Working with temperature (slider + "warm up" button)
- **Exercise 2**: Money calculator (purchases and debts)
- **Exercise 3**: Floor calculator (going up and down)

### 3. 📝 Knowledge Check
- 5 questions with answer options
- Automatic verification
- Detailed analysis of results
- Ability to retake the test

## 🚀 How to run

1. Download all files to one folder
2. Open the `index.html` file in any modern browser
3. Start learning from the "Theory" section

- `index.html` - main HTML page

- **Modern design** with gradients and animations

## 🎓 Educational goals

- Understand what negative numbers are
- Learn to compare negative numbers
- Master addition and subtraction with negative numbers
- Apply knowledge in real situations

## 💡 Usage examples

### Temperature
- Morning -5°C, warmed up by 8°C during the day → became +3°C

### Money
- Have $100, bought a toy for $150 → debt of $50 (-50)

### Floors
- You are on the 2nd floor, go down 5 floors → end up on the -3rd floor (basement)
