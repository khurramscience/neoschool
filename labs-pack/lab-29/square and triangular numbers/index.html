Square S(n) = n²

S(6) = 36
T(n−1) + T(n) = S(n)
2·T(n−1) + S(n) = T(2n−1)
3·T(n) + T(n−1) = T(2n−1)
8·T(n) + 1 = S(2n+1)
How to read the visuals

Each dot is one unit. Triangles are built with rows 1, 2, …, n. A square is n×n. Move the slider n to compare dot counts and verify the identities above.
