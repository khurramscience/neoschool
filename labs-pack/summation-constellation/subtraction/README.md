# 🧮 Laboratory Work: Subtraction

Interactive web page for studying subtraction by schoolchildren. The project includes theory, practice exercises and testing.

## 📚 Functionality

### 1. Theory
- Explanation of the concept of subtraction
- Column subtraction algorithm
- Visual examples
- Special cases (subtraction with borrowing, subtracting zero)

### 2. Practice
- Three difficulty levels:
  - **Easy**: 1-2 digit numbers
  - **Medium**: 2-3 digit numbers  
  - **Hard**: 3-4 digit numbers
- Interactive exercises
- Instant answer checking
- Hints for complex examples
- Performance statistics

### 3. Testing
- 10 questions of varying difficulty
- 3 attempts per question
- Test settings (negative numbers, decimal fractions)
- Detailed results with grading
- Ability to view correct answers

## 🚀 Project Launch

1. Download all project files
2. Open the `index.html` file in any modern browser
3. Start learning from the "Theory" section
