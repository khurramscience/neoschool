# 🎯 Math Game - Sum Sequences

An interactive web game for teaching mathematics to schoolchildren in a playful way. The game helps develop counting skills, logical thinking, and reaction speed.

## 🎮 How to Play

1. **Game Objective**: Find sequences of numbers that sum up to the given target value
2. **Controls**: Click on numbers and drag the cursor horizontally or vertically (not diagonally)
3. **Scoring**: The longer the sequence, the more points you get
4. **Time**: You have 60 seconds to score as many points as possible
5. **Levels**: With each new level, the target sum increases

## 🚀 Running the Game

1. Open the `index.html` file in any modern web browser
2. Click the "Start Game" button
3. Enjoy the game!

## 🎨 Customization

You can easily change game parameters by editing variables in the `script.js` file:

```javascript
this.boardSize = 8;        // Game board size
this.targetSum = 15;       // Initial target sum
this.timeLeft = 60;        // Game time in seconds
```
