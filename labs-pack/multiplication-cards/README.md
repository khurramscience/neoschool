# 🎯 Learn Multiplication - Math Cards

Interactive web application for learning multiplication for school students. The application is created as a mathematics laboratory work.

## 📚 Description

This web application helps students master multiplication through interactive cards with examples. The application includes 5 different difficulty modes, progress bar and results statistics.

## 🎮 Game Modes

1. **Single × Single** - examples like 2 × 3, 7 × 8
2. **Single × 10-19** - examples like 5 × 12, 8 × 17
3. **Single × 10-99** - examples like 6 × 25, 9 × 47
4. **Double × Double** - examples like 23 × 45, 67 × 89
5. **Mixed Mode** - random examples from all previous modes

## ✨ Features

- 🎯 **10 cards in each round** - optimal number for concentration
- 📊 **Progress bar** - visual progress display
- 🟢 **Green color** for correct answers
- 🔴 **Red color** for incorrect answers with correct answer display
- 📈 **Results statistics** - number of correct/incorrect answers and success rate
- 🎨 **Modern design** with gradients and animations
- 📱 **Responsive interface** for mobile devices

## 🚀 How to Run

1. Download all project files
2. Open the `index.html` file in any modern browser
3. Choose a mode and start practicing!

## 🎯 How to Play

1. **Choose a mode** - click on one of the 5 mode buttons
2. **Solve examples** - enter the answer in the field and click "Check" or press Enter
3. **Track progress** - the green bar shows your progress
4. **Get results** - after 10 examples you'll see statistics
5. **Play again** - choose "Play Again" or "New Mode"
