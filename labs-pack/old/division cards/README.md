# Division Cards - Math Learning Lab

An interactive web-based math laboratory for learning division operations through timed card games. Perfect for elementary school students to practice their division skills in a fun and engaging way.

## Features

- **Three Difficulty Levels:**
  - Easy: Division by 2-5
  - Medium: Division by 6-10
  - Hard: Division by 11-15

- **30-Second Timer:** Students race against time to solve as many problems as possible

- **Interactive Cards:** Beautiful card-based interface with clear division problems

- **Real-time Feedback:** Immediate feedback on correct/incorrect answers

- **Score Tracking:** Points system with detailed results at the end

- **Pause/Resume:** Ability to pause the game if needed

- **Responsive Design:** Works perfectly on desktop, tablet, and mobile devices

## How to Use

1. **Open the Application:**
   - Simply open `index.html` in any modern web browser
   - No installation or server setup required

2. **Choose Difficulty:**
   - Select from Easy, Medium, or Hard difficulty levels
   - Each level focuses on different ranges of divisors

3. **Start Playing:**
   - Click "Start Game" to begin the 30-second challenge
   - Solve division problems as quickly as possible
   - Enter your answer and press Enter or click Submit

4. **Game Controls:**
   - **Pause/Resume:** Temporarily stop the timer
   - **End Game:** Finish early and see your results
   - **Enter Key:** Quick answer submission

5. **View Results:**
   - See your final score, accuracy, and total questions answered
   - Choose to play again or return to the main menu
