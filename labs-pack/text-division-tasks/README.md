# 🎓 Laboratory Work: Division

Interactive web page for learning division through interesting stories and tasks for schoolchildren.

## 📚 Description

This web application is a mathematics laboratory work that helps students master division through practical tasks. The application contains 9 different topics, each presented as an engaging story with a mathematical problem.

## 🎯 Features

- **9 diverse topics** with interesting stories
- **Interactive interface** with modern design
- **Progress system** - tracking completed tasks
- **Automatic saving** of progress in browser
- **Hints** for incorrect answers
- **Responsive design** for mobile devices
- **Congratulations** upon completion of all tasks

## 🚀 How to run

1. Download all project files
2. Open the `index.html` file in any modern browser
3. Start learning division!

## 📖 Task topics

1. **🚴 Bicycle Ride** - dividing a group of cyclists
2. **🍕 Pizza for Friends** - dividing pizza among friends
3. **📚 Library** - arranging books on shelves
4. **🎨 Art Studio** - dividing paints among groups
5. **🏀 Sports Ground** - forming teams
6. **🌱 Gardening** - planting seeds in flower beds
7. **🎪 Circus Show** - seating spectators
8. **🚌 School Bus** - seating students
9. **🎁 Birthday Party** - dividing cake among guests

## 🔧 Configuration

To add new tasks, edit the `tasks` object in the `script.js` file. Each task should contain:
- `title` - topic title
- `story` - story/task description
- `question` - task question
- `dividend` - dividend
- `divisor` - divisor
- `answer` - correct answer
