# 🎓 Лабораторная работа: Деление

Интерактивная веб-страница для изучения деления школьниками через интересные истории и задачи.

## 📚 Описание

Это веб-приложение представляет собой лабораторную работу по математике, которая помогает ученикам освоить деление через практические задачи. Приложение содержит 9 различных тем, каждая из которых представлена в виде увлекательной истории с математической задачей.

## 🎯 Особенности

- **9 разнообразных тем** с интересными историями
- **Интерактивный интерфейс** с современным дизайном
- **Система прогресса** - отслеживание выполненных задач
- **Автоматическое сохранение** прогресса в браузере
- **Подсказки** при неправильных ответах
- **Адаптивный дизайн** для мобильных устройств
- **Поздравление** при завершении всех задач

## 🚀 Как запустить

1. Скачайте все файлы проекта
2. Откройте файл `index.html` в любом современном браузере
3. Начните изучение деления!

## 📖 Темы задач

1. **🚴 Велосипедная прогулка** - деление группы велосипедистов
2. **🍕 Пицца для друзей** - деление пиццы между друзьями
3. **📚 Библиотека** - расстановка книг на полках
4. **🎨 Художественная мастерская** - деление красок между группами
5. **🏀 Спортивная площадка** - формирование команд
6. **🌱 Садоводство** - посадка семян в клумбы
7. **🎪 Цирковое представление** - рассадка зрителей
8. **🚌 Школьный автобус** - рассадка учеников
9. **🎁 День рождения** - деление торта между гостями

## 🔧 Настройка

Для добавления новых задач отредактируйте объект `tasks` в файле `script.js`. Каждая задача должна содержать:
- `title` - заголовок темы
- `story` - история/описание задачи
- `question` - вопрос к задаче
- `dividend` - делимое
- `divisor` - делитель
- `answer` - правильный ответ
